2. CSS inline Otra forma de utilizar estilos css es directamente sobre los elementos html, mediante el atributo style dentro de las propias etiquetas html. Dentro del atributo style de un elemento html podemos agregar todos los conjuntos de reglas css necesarios. La principal desventaja de este método es que se mezcla el código html con el código css dificultando su lectura.
